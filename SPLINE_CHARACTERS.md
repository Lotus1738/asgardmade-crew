# AsgardMade Pantheon — Spline 3D Character Guide
## Style Reference: axial.studio — colorful rounded cartoon 3D figures in themed rooms

---

## GLOBAL STYLE RULES (apply to all 10 gods)

**Character anatomy:**
- Body: chunky rounded cylinder torso, no visible neck, head is ~40% of total height
- Arms: short stubby tubes, slightly raised at sides (confident pose)
- Legs: short stumps, slightly apart
- Hands: round mitten-shapes, 3 visible finger bumps
- Eyes: large oval white with small colored iris + black dot pupil
- No nose — just two tiny round dots
- Mouth: simple curved line (slight smile by default)
- Material: smooth matte plastic feel, no texture maps needed
- Outline: add a thin dark edge bevel on all body parts (gives cartoon pop)

**Room setup:**
- Floor: large rounded-corner square slab, 2-3 units thick
- Walls: two L-shaped panels meeting at back corner (not a full box)
- Ceiling: none needed — leave open top
- Props: 2-4 themed objects, all rounded/chunky, same cartoon style
- Lighting: one key light from upper-left, one soft fill from right, ambient warm base
- Camera: slight isometric angle (30° tilt), looking slightly down into the room
- Export: use Spline's "Export → Embed code" for each scene

## HOW TO BUILD IN SPLINE

**Setup order for each god:**
1. New scene → set background to black or dark
2. Build the room first: floor slab → two wall panels → props
3. Add the character last (easier to position in a furnished room)
4. For the character body: start with a cylinder (round cap ends), scale to chunky proportions
5. Duplicate for each body part, parent to a group called "CHARACTER"
6. Add materials: use Spline's standard material, set to flat/matte, pick the main color
7. For glow effects: use Spline's "Bloom" post-processing + set object emission to the color

**Animations to add (keep them subtle):**
- Character: idle bob — move Y up 0.05 units, ease in-out, loop 2s
- Floating props: gentle rotation, 0.2 degrees/frame
- Glow elements: pulse emission from 0.8 to 1.0 opacity, loop 3s

**Embed into HUD:**
1. In Spline: File → Export → Embed Code → copy the `<script>` + `<spline-viewer>` tags
2. In `public/index.html`: add a `<div class="god-room" id="room-odin">` section per god
3. Paste the embed code inside
4. Size the container: `width: 300px; height: 300px` works well in a grid layout
5. Each scene gets its own URL from Spline — they load independently

**Performance tip:** Keep each scene under 50 objects. Spline embed loads are async so multiple scenes on one page won't block each other.
